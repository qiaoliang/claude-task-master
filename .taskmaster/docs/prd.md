# 简单计算器应用

## 项目概述
创建一个简单的命令行计算器应用，支持基本的数学运算（加、减、乘、除）。

## 功能需求
1. 接受用户输入的两个数字和运算符
2. 执行相应的数学运算
3. 显示计算结果
4. 处理除数为零的错误
5. 提供友好的用户提示

## 技术栈
- 语言: JavaScript (Node.js)
- 运行时: Node.js
- 类型: 无需框架，纯 Node.js

## 用户界面
命令行界面 (CLI)

## 预期输出示例
```
欢迎使用简单计算器！

请输入第一个数字: 10
请输入运算符 (+, -, *, /): +
请输入第二个数字: 5

计算结果: 10 + 5 = 15
```
